# 1) MarkDown是什么
一种用纯文本形式写排版的标记语言，常用于:REMADE,文档,笔记,README,Github Issue等描述。

目标是:写起来快,读起来容易,可以转化为PDF。

---
# 2)最常用的语法
## 2.1标题
用#表示标题级别:
```
# 一级段落
## 二级段落
### 三级段落
#### 四级段落
```

## 2.2段落与换行
空一行才会分段
```
这是第一段

这是第二段
```

## 2.3加粗、斜体、删除线
**加粗**
*斜体*
~~删除线~~
```
**加粗**
*斜体*
~~删除线~~
```

## 2.4列表(有序/无序/任务列表)
- 无序 1
- 无序 2
    - 子项(缩进两个空格和四个空格都常见)

1. 有序 1
2. 有序 2

- [ ] 未完成
- [x] 已完成

```
- 无序 1
- 无序 2
  - 子项（缩进两个空格或四个空格都常见）

1. 有序 1
2. 有序 2

- [ ] 任务未完成
- [x] 任务已完成e
```
## 2.5引用
> 这是一段引用
