## 🌟 项目愿景
通过较为系统化的整理，巩固专业基础，记录从零到一的计科学习历程，并为有需要的同学提供参考。

---

## 📚 课程导航

这里列出了目前已归档的课程内容：

### 1.程序设计(Programming)
- **目录结构**:`Programming/`
- **核心内容**:
    - 📖 **CG practice**:CG上的部分练习解答
    - 📁 **Library_system**:课程大作业源代码和测试集
    - 🙋‍♂️ **小班测试、资料**:包含测试解答和小班PPT

### 2. 算法设计与分析 (Algorithm Design and Analysis)
- **目录结构**: `The design and annlysis of algorithm/`
- **核心内容**:
    - 🖥️ **Code**:包含算法刷题代码
    - 📂 **Experiment**:包含实验代码、实验报告
    - 📝 **HomeWork**:课程作业代码、报告、测试集
    - 🙋‍♂️ **小班讨论**:包含小班讨论PPT、源代码

### 3. 数字电路逻辑设计与分析(Digital circuits and logical programming)
- **目录结构**:`Digital circuits and logical programming/`
- **核心内容**:
    - 🖥️ **Discission**:包含小班讨论PPT和电路设计
    - 📂 **Experiment**:包含三次实验设计、实验报告,以及模型机电路设计、报告

### 3.数据结构(Data structure)
- **目录结构**:`Data structure/`
- **核心内容**:
    - **Code**:包含学习线性、树、图数据结构的源代码
    - **笔记**:学习时的markdown笔记
---

## 🛠️ 环境配置
仓库中的代码主要基于以下环境：
- **语言**: C++ (C++11 或更高)
- **编译器**: GCC / MinGW
- **编辑器**: VS Code / Dev-C++
- **构建工具**: CMake (可选)

---

## 📂 目录规范
为了方便查找，本仓库遵循以下命名约定：
- `Experiment/` - 课程相关实验及报告
- `Homework/` - 课后作业题
- `Discission` - 小班讨论资料
